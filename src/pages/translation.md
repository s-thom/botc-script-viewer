Hello! Thank you for taking interest in translating this site. This is the first project I've done where I've tried to support more languages than just English, and I want to do it well. Unfortunately for both you and me, I am only fluent in English.

This website isn't as simple as having one big list of every bit of text that appears on the site. Instead, the words you read come from different places. This is hard to explain in a short space like the footer of every page, so I instead wrote this page to explain it.

## Character and Night Order descriptions

All text for characters and night order comes from one of two places:

1. [The official translations on Weblate](https://translation.botc.app/).
2. [The community spreadsheet](https://docs.google.com/spreadsheets/u/0/d/1aAJdqSTafHnw01w-WZ94UPx1Me70Kz-EG1NFfBht2tA/htmlview).

If you want to improve the character names, character descriptions, night order descriptions, or jinxes, the best place to do that is to contribute to [the official translations](https://translation.botc.app/). That way your improvements will help everyone playing Blood on the Clocktower, not just people using this website.

You do not need an account on the site to submit suggestions, but if you're going to translate many components then you might want to create one to help coordinate with other translators. You can read more about how to translate on the Overview tab of the [BOTC App project](https://translation.botc.app/projects/botc-app/).

## Other text on this website

As I was adding translation support to this website I was surprised by the amount of text I had added without even thinking about it. Even such a simple website like this has a lot going on. I wanted to at least provide some basic translation as a starting point, so I used [Kagi Translate](https://translate.kagi.com/) to translate the most important components. I know that this is not perfect, and I need help in order to make it better.

The full list of components that are part of this website specifically can be found in the source code at [src/i18n/en.json](https://github.com/s-thom/botc-script-viewer/blob/main/src/i18n/en.json). You can contribute to this website by [creating an issue on GitHub](https://github.com/s-thom/botc-script-viewer/issues), or [submitting a pull request](https://github.com/s-thom/botc-script-viewer/pulls) if you know how.
